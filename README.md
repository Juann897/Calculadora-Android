# 📱 Calculadora Simples - Android

## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de uma calculadora simples para Android, criada como atividade prática da disciplina de Programação.

O aplicativo permite realizar operações matemáticas básicas utilizando uma interface simples e intuitiva.

---

## ⚙️ Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão
* 🧹 Limpar dados
* ⚠️ Tratamento de erro (divisão por zero)

---

## 🧠 Lógica utilizada

A aplicação foi desenvolvida utilizando:

* Variáveis para armazenar os números digitados
* Estrutura condicional `when` para realizar as operações
* Conversão de dados de `String` para `Double`
* Controle de fluxo para evitar erros de cálculo

---

## 🖥️ Tecnologias utilizadas

* Kotlin
* Android Studio
* XML (Interface)

---

## 📂 Estrutura do projeto

* `MainActivity.kt` → Lógica principal da aplicação
* `activity_main.xml` → Interface gráfica

---

## 🚀 Como executar

1. Abrir o projeto no Android Studio
2. Conectar um dispositivo ou iniciar um emulador
3. Executar o projeto
